# Big-Data-Analysis
Laboratory works and assignments for the Big Data Analysis course (2025–2026) at Beijing Institute of Technology. Covers data crawling, AI-enhanced ETL, analytics pipelines, FinBERT, knowledge graphs, and multi-agent systems.

## Laboratory Works
| № | Lab | Description |
|---|---|---|
| 1 | [Data Crawler & Cleaner](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/Data_Crawler) | Web data crawling and cleaning using a multi-agent pipeline |
| 2 | [CLIP Image–Text Matching](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/CLIP) | Image classification and semantic matching using contrastive learning (CLIP) |
| 3 | [AI-Enhanced ETL (PDF Data Extraction)](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/ETL) | AI-enhanced ETL pipeline for extracting structured data and comments from PDF documents |
| 4 | [Autolabelling_Snorkel](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/Autolabelling_Snorkel) | Weak supervision and automatic data labeling using Snorkel, Label Studio, and TF-IDF + Logistic Regression |
| 5 | [Regression and Clustering Experiments](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/Regression_Clustering) | Comparative analysis of regression and clustering methods on synthetic and real-world datasets |
| 6 | [Financial Sentiment Analysis (BERT vs FinBERT)](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/FinBERT) | Experimental comparison of general-purpose BERT and domain-specific FinBERT for financial sentiment analysis |
| 7 | [Data_parallelism_and_Model_Parallelism](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/Data_parallelism_and_Model_Parallelism) | Distributed deep learning architectures: PyTorch DDP, TorchDistributor, Ray Train |
| 8 | [Reccomendation_System_Assigment](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/Reccomendation_System_Assigment) | Comparative study of collaborative filtering and LLM-enhanced recommendation systems |
| 9 | [LLM_Financial_Knowledge_Graph_Pipeline](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/LLM_Financial_Knowledge_Graph_Pipeline) | Construction of a financial knowledge graph using LLM-based entity and relation extraction with Neo4j |
| 10 | [AI_Agents_and_AutoGPT_Experiment](https://github.com/BaldursFrog/Big-Data-Analysis/tree/main/AI_Agents_and_AutoGPT_Experiment) | Exploration of AI agent architectures and AutoGPT workflow with a simulated autonomous data analysis experiment using the Titanic dataset |
